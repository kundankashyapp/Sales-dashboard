# Sales-dashboard
Power BI E-commerce Dashboard is to understand sales performance 

The objective of this Power BI E-commerce Dashboard is to understand sales performance in an easy way. 
It shows total sales, month-wise sales, top customers, and popular payment modes. 
This dashboard helps to track business growth, understand customer behavior, and support better and faster business decisions.
